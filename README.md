# Atlas Documentation

Developer documentation for the Atlas deterministic sensor infrastructure platform.

This repository hosts the documentation site for Atlas, including:

- platform overview
- Fusion hardware architecture
- Fusion firmware overview
- DSIL SDK
- ROS2 integration
- evaluation kit setup
- certified sensor guidance

---

## Atlas Platform

Atlas is a deterministic sensor infrastructure for robotics.

Platform stack:

Sensors → Fusion Hardware → DSIL SDK → ROS2 / Robot Applications

### Core Repositories

- [fusion-hardware](https://github.com/sensordeck/fusion-hardware)
- [fusion-firmware](https://github.com/sensordeck/fusion-firmware)
- [dsil-sdk](https://github.com/sensordeck/dsil-sdk)
- [ros2-dsil](https://github.com/sensordeck/ros2-dsil)
